Clutch

本地 AI 多 Agent 编排与监督控制台

在 Claude Code / Cursor 之上加一层可持久化、可观测、可编辑的流程控制层。零代码拖拽 SOP，LangGraph 调度本地 CLI 与大模型，统一工作台全程监督与人工审批。

核心能力

  • 可视化工作流编排
  • 本地 AI 工具接入
  • 统一监督控制台
  • 人机协同门控

Tauri 2 · React 19 · FastAPI + LangGraph · MIT · macOS
