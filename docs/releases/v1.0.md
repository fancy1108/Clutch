# Clutch v1.0 — Product Snapshot

**Released:** 2026-06-29  
**Tag:** [v1.0.0](https://github.com/fancy1108/Clutch/releases/tag/v1.0.0)  
**Changelog:** [CHANGELOG § 1.0.0](../../CHANGELOG.md#100---2026-06-29)

## What this version is

Clutch v1.0 is the **first public release** of a local desktop app for supervising multi-agent AI workflows. Users author zero-code Flows, run real CLI/MCP tools on an authorized workspace, approve risky actions, and inspect Chat, Terminal, and file changes in one window.

Distribution: **unsigned macOS DMG** (Apple Silicon) via GitHub Releases, plus build-from-source instructions.

## Shipped capabilities

- Tauri desktop shell + FastAPI/LangGraph sidecar on `localhost`
- Visual workflow editor and template library (e.g. video production, weather-to-vision)
- Real-time WebSocket supervision (Chat, Terminal, Flow highlight, validation cards)
- Human gates: Approve / Reject / Retry / reassign to Builder
- Hybrid Runtime: persistent shell sessions for plain chat and Flow refine (`@Agent`, `/continue`)
- Workspace picker, Files panel, Changes/Diff, run history, session persistence
- Models UI: multi-provider, CC Switch import, Ollama, Keychain-backed API keys
- MCP configuration, Skills registry, theme + EN/ZH i18n
- First-run onboarding wizard (7 screens)
- Install, privacy, build-from-source, and security disclosure docs

## Not included / known limits (v1.0.0)

- No Apple notarization; Gatekeeper workaround documented in INSTALL
- No Windows/Linux installers
- No in-app auto-update
- Debug API not yet restricted; WebView CSP not hardened (OSR-16)
- Sidecar PyInstaller bundle still has console window (OSR-17)
- OS-level sandbox and fine-grained exec policy not shipped (see `memory/BACKLOG.md`)
