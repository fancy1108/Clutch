# Clutch v1.1.0 — Product Snapshot

**Released:** 2026-07-03  
**Tag:** [v1.1.0](https://github.com/fancy1108/Clutch/releases/tag/v1.1.0)  
**Changelog:** [CHANGELOG § 1.1.0](../../CHANGELOG.md#110---2026-07-03)

## What this version is

Clutch v1.1.0 is a **minor release** centered on **Terminal Orchestra (D34)** — embedded multi-lane CLI terminals, Orchestrator dispatch, and handoff supervision in Single Agent sessions — plus **Windows desktop polish**, **CodeBuddy CLI**, **OpenCode Zen**, **Agnes Video V2.0**, and agent-scoped Settings tabs. macOS remains the primary tested platform; Windows installers are CI-built ([#23](https://github.com/fancy1108/Clutch/issues/23) still open for physical smoke).

## Shipped capabilities

- **Terminal Orchestra:** Chat ↔ Terminal toggle; multi-lane xterm PTY; OrchestratorBar `@` dispatch; handoff files; Overview dispatch log; lane collapse with PTY keepalive; leave-session history; CLI resume commands.
- **Orchestrator → PTY inject:** Dispatched tasks echo into target lane and auto-submit Enter.
- **Terminal input dock:** OrchestratorBar matches Chat fixed bottom layout with measured gap above input.
- **Windows polish:** Sidebar collapse, font-size preference, session switch caching, background WebSocket ([#28](https://github.com/fancy1108/Clutch/pull/28)).
- **CodeBuddy CLI:** Headless routing + session resume.
- **OpenCode Zen:** Five free chat models in Settings → Models (Clutch Agent tab).
- **Agnes Video V2.0:** Chat video generation with inline playback and download.
- **Agent-scoped Settings:** Models / MCP / Skills tabs per agent type.
- All v1.0.3 capabilities (Hybrid shell pool queue, OpenCode CLI routing, brand refresh, Ollama fix, …).

## Not included / known limits (v1.1.0)

- **Terminal Orchestra** is Single Agent plain chat only — workflow sessions do not show Chat/Terminal toggle.
- Rivet / Aider / Ollama have no UUID session resume in copy-paste cards (honest fallbacks only).
- macOS DMG still **unsigned** (Gatekeeper workaround in INSTALL).
- **No in-app auto-update on Windows** — manual reinstall from Release.
- Windows installers: CI-built, **not maintainer smoke-tested on physical Win10/11** at ship time ([#23](https://github.com/fancy1108/Clutch/issues/23)).
