# Clutch v1.0.3 — Product Snapshot

**Released:** 2026-07-01  
**Tag:** [v1.0.3](https://github.com/fancy1108/Clutch/releases/tag/v1.0.3)  
**Changelog:** [CHANGELOG § 1.0.3](../../CHANGELOG.md#103---2026-07-01)

## What this version is

Clutch v1.0.3 is a **minor release** focused on **Hybrid plain-chat reliability** and **day‑2 polish**: cross-session **shell pool queueing**, **OpenCode CLI** routing, refreshed **brand / app icons**, clearer **CLI error copy**, **Ollama model list sync** in Settings, and a **real-connection E2E acceptance** suite for maintainers. macOS remains the primary tested platform; Windows installers are CI-built ([#23](https://github.com/fancy1108/Clutch/issues/23) still open for physical smoke).

## Shipped capabilities

- **Hybrid shell pool queue (plain chat):** When `CLUTCH_SHELL_MAX_SESSIONS` is saturated, new Hybrid CLI turns enter a **global FIFO queue** with in-input **blocker list + agent avatars**; auto-resume when a slot frees (no Supervisor reject).
- **Same-session message queue:** Send while a turn runs; messages show as **待发送消息** and drain in order.
- **OpenCode CLI:** First-class headless routing for `opencode-cli` agents (Hybrid when enabled).
- **Built-in Agnes 2.0 Flash** chat model preset.
- **Settings → Models (Ollama):** Lists **locally installed** `ollama list` tags; stale `active_model_id` from another machine auto-falls back.
- **CLI errors (#19):** Single-layer failure copy; clearer gateway-busy message on 529/5xx.
- **Brand refresh:** New Clutch mark assets and regenerated Tauri icon set.
- **Install & docs:** Bilingual README, [`GETTING_STARTED`](../../GETTING_STARTED.md), curl + Homebrew tap install paths.
- All v1.0.2 capabilities (Windows MSI/NSIS, macOS in-app update, Rivet, Hybrid runtime, Flow editor, …).

## Not included / known limits (v1.0.3)

- **FLOW / Flow Refine** do not use the global shell pool queue — only **plain-chat Hybrid CLI** (Claude / Codex / Aider / Antigravity) queues cross-session.
- macOS DMG still **unsigned** (Gatekeeper workaround in INSTALL).
- **No in-app auto-update on Windows** — manual reinstall from Release.
- Windows installers: CI-built, **not maintainer smoke-tested on physical Win10/11** at ship time ([#23](https://github.com/fancy1108/Clutch/issues/23)).
- Loop integration still deferred.
