# Clutch v1.0.2 — Product Snapshot

**Released:** 2026-07-01  
**Tag:** [v1.0.2](https://github.com/fancy1108/Clutch/releases/tag/v1.0.2)  
**Changelog:** [CHANGELOG § 1.0.2](../../CHANGELOG.md#102---2026-07-01)

## What this version is

Clutch v1.0.2 is a **minor release** expanding distribution and day‑2 operations: **Windows installers**, **macOS in-app auto-update**, a **macOS sidecar lifecycle fix (#18)**, and **expanded CLI tool discovery** (including Rivet). macOS remains the primary tested platform; Windows packages are CI-built with automated tests but not yet fully smoke-tested on physical hardware by maintainers.

## Shipped capabilities

- **macOS DMG** (Apple Silicon) via GitHub Releases — unchanged unsigned distribution model
- **Windows MSI + NSIS** (`Clutch_1.0.2_x64_en-US.msi`, `Clutch_1.0.2_x64-setup.exe`) on the Release page
- **In-app updates (macOS, OSR-20):** `latest.json` + signed `.tar.gz` bundles; update banner in the desktop app
- **Sidecar lifecycle (#18):** kill stale orchestrator on launch; terminate sidecar on quit; Dock reopen restores main window
- **Rivet CLI + expanded tools whitelist:** curated install hints; Rivet PATH fix for nvm/npm shebangs in packaged app
- All v1.0.0 / v1.0.1 capabilities (Hybrid Runtime, Flow editor, onboarding, Keychain keys, etc.)

## Not included / known limits (v1.0.2)

- macOS DMG still **unsigned** (Gatekeeper workaround in INSTALL)
- **No in-app auto-update on Windows** — manual reinstall from Release
- **v1.0.0 / v1.0.1 users** must manually install v1.0.2 once; only then can macOS in-app updates apply
- Windows installers: CI-built, **not maintainer smoke-tested on physical Win10/11** at ship time
- No Apple notarization; no Linux installers
- Loop integration deferred to **v1.0.3**
